# Regex-Engine

## Rust install

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

rustc --version  
cargo --version  
```


## Build and run
```bash
# Compile
cargo build

# Demo 
cargo run

# Tests
cargo test
cargo test -- --nocapture
cargo test test_ab_star
cargo test test_star
```
